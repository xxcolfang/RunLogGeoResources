# RunLogGeoResources

Public offline geo resources for RunLog.

## Files

- `Resources/RunLogGlobalPeaks.sqlite.zip`
  - Global OSM peak database for first-run download.
  - Unzip before opening as SQLite.
  - Source: `global-osm-peaks.geojson`
  - Contains about 1.08M peak records.

## Release Assets

- `RunLogPeaks.sqlite`
  - China OSM peak database.
  - Stored as a GitHub Release asset under the `geo-resources` tag.
